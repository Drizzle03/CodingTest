1. 메모리 크기
대체로 코딩테스트에서는 128MB~512MB로 메모리를 제한함

데이터 개수 | 메모리 사용량
1,000 -> 약 4KB
1,000,000 -> 약 4MB
10,000,000 -> 약 40MB

2. 연산 속도
파이썬의 경우 1s -> 2,000만번 연산 수행

2-1. 연산 속도와 시간복잡도
데이터의 개수가 100만일 때 O(NlogN)은 약 2천만회를 계산함.
즉, 데이터 개수가 100만이고, 시간제한이 1s일 때
-> NlogN 이내의 알고리즘을 사용할 것

2-2. 시간복잡도 계산법
컴퓨터에서는 이진법을 사용하므로 로그의 밑을 2로 둠. 즉, Nlog2N임.
만약 데이터 개수가 100만이라면 2^20 = 1,048,576이므로
100만은 2^20의 근삿값. 
1,000,000 * 20 = 20,000,000임
